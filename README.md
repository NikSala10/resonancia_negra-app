**Resonancia Negra** is a cooperative–competitive narrative game set in a post-apocalyptic world invaded by mysterious creatures known as *Shadows*. Players take on different survivor roles and must face dangerous scenarios through **group decisions, strategic discussions, and individual actions**.

The game is designed to create a **hybrid dynamic between cooperation and competition**, where players must collaborate to survive while also trying to achieve the best individual performance.

---

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

# 🎮 Overview

In *Resonancia Negra*, players form a group of survivors navigating a dangerous world. Throughout the game they encounter narrative events and challenges that require them to **debate, negotiate, and make decisions together**.

However, not every decision benefits everyone equally. Players accumulate **individual points**, creating moments where personal interests may conflict with the group's survival.

This balance between **teamwork and rivalry** is the core experience of the game.

# 🔄 Game Dynamics

The game produces two types of gameplay dynamics:

### 🔗 Dependent Dynamics (Cooperation)

Players rely on each other to progress.

Examples:
- Coordinating strategies to overcome group challenges
- Making collective decisions that affect everyone
- Sharing responsibility for survival

### 🧍 Independent Dynamics (Competition)

Players also pursue personal success.

Examples:
- Maximizing individual score
- Choosing actions that benefit their character
- Competing for the best final performance

These dynamics create a **balanced tension between collaboration and rivalry**.

# 🔍 Key Findings

Playtesting revealed that:

- Players cooperate more when rewards benefit the entire group.
- Competition increases when decisions provide individual advantages.
- Some roles promote more cooperative behavior than others.
- Strategic discussions strongly influence group decisions.

Overall, the game creates a **hybrid experience combining cooperation and competition**.
